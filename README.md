# Pic-Tac-Toe

## Coded in Python

### Console-based Tic-Tac-Toe Game

### Features

1. Multiplayer Mode
2. Human VS Computer Mode
3. Score Recording
4. Replay Option

### How to test it?

1. Simply run the .exe (execution) file.
2. If it fails, open Command Prompt in the .exe file directory and run using Command Prompt.
